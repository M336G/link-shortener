# link-shortener
An easy to use API to create and manage short links

## How to setup?
1. Clone the repository with `git clone https://github.com/M336G/link-shortener.git`.
2. Install [Bun](https://bun.sh/) and run `bun install`.
3. Run the project with `bun run start`.

## Configuring
See the notes in [.env.example](https://github.com/M336G/link-shortener/blob/main/.env.example) if you want to configure your link shortener server further.
